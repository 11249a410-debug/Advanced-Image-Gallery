# 🌸 Nature Image Gallery

A modern and responsive Image Gallery built using HTML, CSS, and JavaScript. This project showcases beautiful nature and flower images with advanced features such as Dark/Light Mode, Search Functionality, Category Filters, Favorites, Download Options, Lightbox Preview, and Animated Flower Background.

---

## 🚀 Features

### Core Features
✅ Responsive Image Gallery Layout

✅ Hover Effects and Smooth Animations

✅ Lightbox Image Preview

✅ Mobile-Friendly Design

### Advanced Features
🌙 Dark / Light Mode Toggle

🔍 Search Images by Name

🏷️ Category Filtering

❤️ Favorite Images

📥 Download Images

🌸 Animated Flower Background

✨ Glassmorphism UI Design

📱 Fully Responsive Layout

⚡ Smooth Transitions and Effects
